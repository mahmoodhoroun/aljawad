## Dynamic Filters for List Views in Odoo

What if you could configure custom filters in list view of a model so that users
can easily filter records for various scenarios? This module helps to achieve
exactly that!


https://github.com/user-attachments/assets/2d219229-d1b8-49d9-878a-52b747dcfc60

## Usage

1. Install this module.
2. Ensure that the user has admin rights.
3. Go to Settings -> Technical -> User Interface -> List Filters.
4. Configure the filters as needed for different models.
