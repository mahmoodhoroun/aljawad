Direct Print Report
===================

This module allows users to print reports directly from Odoo without the need to download them as PDFs. It provides a seamless and efficient way to access printed reports instantly.

Features
--------

- No additional software required: Works with your existing setup. Just ensure the printer driver is installed on your PC. No need for external print services or complex setups.
- Preview before printing: View the document before printing to avoid errors and reduce unnecessary print waste.
- Direct print without IoT box: Sends reports directly to the printer, saving costs on additional hardware.
- Out-of-the-box functionality: Easy setup with minimal configuration. Install the module, configure reports, and start printing.
- Customizable report settings: Choose which reports to print directly and which to download as PDFs, based on your business needs.
- Compatible with both community and enterprise editions: Ensures seamless operation regardless of the Odoo edition.
- Direct print option: Shows a popup for direct printing whenever a report is generated.
- Conditional direct print: Offers flexibility to enable direct printing only for selected reports.

Screenshots
-----------

1) Settings Direct Print:  
   If enabled, a popup appears for direct printing when a report is generated.  
   Navigation: Settings → Direct Print  

2) Settings view:  
   Under the technical menu, users can configure the reports.  
   Navigation: Settings → Technical → Actions → Reports  

3) Settings Reports view:  
   Users can select specific reports to enable the direct print feature.  

4) Reports Form view:  
   Users can enable the direct print option for selected reports.  

5) Print view:  
   Direct print feature for related reports.  
   Navigation: Sales → Quotation → Print  

6) Inventory Picking Operation view:  
   Navigation: Inventory → Operations → Picking  

7) Picking Operation print view:  
   Enables direct printing for inventory operations.
